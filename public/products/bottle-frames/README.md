Nexovia transparent bottle sequence
===================================

This directory is the alpha-supported replacement for `Nexovia_Bottle_full_vid.mp4`.

- `frame_001.png` through `frame_058.png` are transparent PNG frames.
- The website uses these frames for the hero/product bottle so the original maroon video background never appears.
- Do not place the raw MP4 directly in the hero unless it is replaced by an alpha-supported export such as `public/products/nexovia-bottle-alpha.webm`.

If the sequence is regenerated, preserve transparent pixels around the bottle and keep the same `frame_###.png` naming pattern.
